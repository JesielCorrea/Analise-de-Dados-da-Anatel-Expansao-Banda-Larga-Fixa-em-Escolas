# 📊 anatel-banda-larga-escolas-br

![Dashboard](imagens/visao-geral.png)

## 📌 Sobre o Projeto

Projeto desenvolvido em **Power BI** utilizando dados públicos da **Anatel** com o objetivo de analisar a expansão da banda larga fixa nas escolas brasileiras entre 2008 e 2018.

O dashboard apresenta indicadores sobre cobertura, distribuição por estados, concessionárias e tecnologias utilizadas, permitindo identificar padrões e apoiar análises sobre a evolução da conectividade no período.

## 💡 Principais Insights

- Minas Gerais apresentou o maior número de municípios atendidos no período analisado.
- Bahia e São Paulo aparecem na sequência entre os estados com maior cobertura.
- A distribuição evidencia concentração do atendimento em alguns estados.
- A tecnologia **Rádio** representou aproximadamente **67%** das conexões registradas, seguida pela **Fibra** com cerca de **23%**, evidenciando o predomínio dessas soluções na expansão da conectividade das escolas.
