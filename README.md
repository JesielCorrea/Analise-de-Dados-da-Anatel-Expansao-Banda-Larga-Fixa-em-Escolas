# 📊 Panorama de Expansão da Banda Larga Fixa nas Escolas (2008–2018)

## 🎯 Objetivo

Analisar a expansão da banda larga fixa nas escolas brasileiras utilizando dados públicos da **ANATEL**.

## 🛠 Ferramentas

- Power BI
- Power Query
- DAX
- Excel
- ETL

## 📈 Indicadores

- Total de Estados Atendidos
- Total de Municípios Atendidos
- Total de Concessionárias
- Municípios por Concessionária

## 💡 Principais Insights

- A **Telemar/MG** apresentou o maior número de municípios atendidos.
- A cobertura concentrou-se em poucas concessionárias.
- A tecnologia utilizada varia conforme a região.
